Реализуйте функцию, которая принимает на вход массив, состоящий из массивов-пар и возвращает объект, полученный из этих пар. fromPairs([['fred', 30], ['barney', 40]]); // > { 'fred': 30, 'barney': 40 }

Если при конструировании объекта попадаются совпадающие ключи, то берётся ключ из последнего массива-пары. fromPairs([['cat', 5], ['dog', 6], ['cat', 11]] // > { 'dog': 6, 'cat': 11 }
